==============================

![](images/snapshot.png)


共有两套输入法引擎

- 基于rime的实现：跨平台，重点优化，推荐使用
- 基于ibus的实现：智能化较少，只能linux下使用


码文方面，共有两份方案

- mn-cn mongolian-unicode-cn 蒙文回鹘式，内蒙用的
- mn-mn mongolian-cyrillic 蒙文西里尔式，外蒙用的

使用方法详见各自文件夹的说明。
